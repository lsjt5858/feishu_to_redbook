# 飞书转小红书格式插件

一个浏览器插件，可以将飞书文档内容快速转换为小红书风格的格式。

## 功能特点

- ✨ 自动提取飞书文档内容
- 🎯 转换为小红书风格格式（添加emoji、优化排版）
- 📋 一键复制到剪贴板
- 🎨 精美的用户界面

## 安装方法

### Chrome/Edge 浏览器

1. 打开浏览器扩展管理页面
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

2. 开启"开发者模式"（右上角开关）

3. 点击"加载已解压的扩展程序"

4. 选择本项目文件夹

5. 插件安装完成！

## 使用方法

1. 打开飞书文档页面

2. 点击浏览器工具栏中的插件图标

3. 点击"转换当前页面"按钮

4. 查看转换结果，点击"复制到剪贴板"

5. 粘贴到小红书编辑器中

## 转换效果

插件会自动：
- 为标题添加精美emoji装饰
- 优化段落排版
- 添加小红书风格的结尾语
- 保持内容的可读性

## 项目结构

```
.
├── manifest.json       # 插件配置文件
├── popup.html         # 弹窗界面
├── popup.js           # 弹窗逻辑
├── content.js         # 内容脚本
├── converter.js       # 转换核心逻辑
├── styles.css         # 样式文件
└── icons/             # 图标文件夹（需自行添加）
```

## 注意事项

- 需要在飞书文档页面中使用
- 首次使用时可能需要刷新页面
- 建议在转换后根据实际需求微调内容

## 开发说明

本插件使用原生 JavaScript 开发，无需额外依赖。

### 自定义转换规则

编辑 `converter.js` 文件中的 `FeishuToXiaohongshuConverter` 类：
- `emojis` 对象：自定义emoji列表
- `convert()` 方法：修改转换逻辑
- `addXiaohongshuEnding()` 方法：自定义结尾语
